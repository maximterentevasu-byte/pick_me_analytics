# Railway advanced Telegram analytics

## Что считает
- Подписчики
- Средний охват поста
- Средний просмотр поста
- ER (по просмотрам) %
- ER (по активностям) %
- Ср. кол-во реакций
- Ср. кол-во комментариев
- Ср. кол-во репостов
- Кол-во постов
- Сумма просмотров постов
- Сумма реакций
- Сумма комментариев
- Сумма репостов
- Engagement на пост
- Реакции на 1000 просмотров
- Репосты на 1000 просмотров
- Комментариев на 1000 просмотров
- Виральность постов %
- Индекс качества контента

## Railway
Variables:
- TG_API_ID
- TG_API_HASH
- TG_STRING_SESSION
- CHANNELS
- GOOGLE_SHEET_ID
- GOOGLE_SERVICE_ACCOUNT_EMAIL
- GOOGLE_PRIVATE_KEY

npm run report
